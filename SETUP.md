# Setup

## Requirements

- Windows 11
- Node.js 22+
- npm

## First run

```bash
npm install
npm run dev
```

Open:

`http://127.0.0.1:4301`

## Build and check

```bash
npm run check
```

This runs the TypeScript build and the Vite production build.

## Install as a PWA

Once the dev or preview server is open in a Chromium browser or Android browser with install support, use the browser install prompt to add the app.

## Local-only planning

Use `.local/` for planning notes, scratch files, and temporary exports that should not be synced to the repo.
